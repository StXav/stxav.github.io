
# Serenity Wellness – Static Site

A clean, single‑page wellness website (massage & body treatments) ready for GitHub Pages.

## Quick start
1. Create a public repo named `your-username.github.io` or any project name.
2. Upload these files (or `git push`).
3. In **Settings → Pages**, set Source to `Deploy from a branch`, choose `main` and folder `/ (root)`.
4. Optional: replace `assets` images and update contact details in `index.html`.
